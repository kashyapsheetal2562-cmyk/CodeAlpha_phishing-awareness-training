# Phishing Awareness Training

## 📌 Purpose
This project educates users about phishing attacks, fake websites, and social engineering tactics.  
It provides real-world examples, best practices, and interactive quizzes to help individuals avoid falling victim.

## ⚙️ Contents
- Presentation slides (PDF/PowerPoint)
- Examples of phishing emails and websites
- Interactive quiz for engagement
- Best practices checklist

## ▶️ Usage
- Open the `presentation/` folder to view training slides.
- Review `examples/` for real-world phishing cases.
- Take the quiz in `quiz.md` to test your knowledge.
